# Roadmap

## Phase 1: Foundation

- Monorepo scaffold.
- Local database, cache, and object storage.
- API health endpoint.
- Authentication skeleton.
- Listings skeleton.
- Web landing page.
- Mobile home screen.

## Phase 2: Accounts and trust

- Registration and login.
- Phone and email verification.
- User profile.
- Seller verification status.
- Trust and safety indicators.

## Phase 3: Listings

- Category management.
- Listing creation and editing.
- Multiple images per listing.
- Location-based search.
- Saved listings.

## Phase 4: Messaging and offers

- Buyer-seller conversations.
- Offer creation.
- Offer acceptance/rejection.
- Transaction records.

## Phase 5: Payments and moderation

- Payment provider selection by country.
- Protected checkout.
- Disputes.
- Admin moderation dashboard.
